# 🌐 Network Traffic Analyzer & Anomaly Detection

## 📌 Overview
This project is a real-time Network Traffic Analyzer that captures live packets, analyzes them, and detects anomalies using Machine Learning.

## 🚀 Features
- 📡 Real-time packet capture (Scapy)
- 📊 Live dashboard (Flask + Chart.js)
- 🤖 ML-based anomaly detection (Isolation Forest)
- 🔐 Login system
- 📄 PDF report generation
- 📈 Protocol visualization (TCP/UDP)

## 🛠️ Tech Stack
- Python
- Flask
- Scapy
- Pandas
- Scikit-learn
- Chart.js
- ReportLab
